Fitur Universal
- Full CRUD (Simple Edit Langsung by Dropdown ataupun Detail Aksi dari luar halaman tanpa perlu klik detail)
- Fitur filter & sortir (All Time,Yearly,Monthly,Weekly,Daily,Custom) : Jika di klik / ubah = Datanya otomatis berubah realtime
- Fitur Icon / Logo Salin DiSamping Teks : Disemua Element Penting - Semi Penting - Detail Informasi.
- Fitur Sortir A-Z atau Z-A : Disetiap Header & Kolom Informasi Halaman Jika Di Klik (Contoh : Halaman Transaksi, Users, dan lainnya)
- Fitur semua halaman / kolom : Jika di klik = Mengarah langsung ke halaman informasi Utama product tersebut.
Contoh : Item Terjual = Mengarah ke Transaksi , Registered User = Mengarah ke Member , Total Pendapatan = Ke Transaksi , Total Produk = ke Halaman produk dan lainnya.


Berikut adalah rincian desain utamanya

Dashboard 
- Full Informasi : Total Item Terjual , Total Produk , Total Order , Total Member , Total Pendapatan , Top Kategori , Top Product (Best Seller) , Transaksi Terakhir , Recently Registered User + Sortir Real Time Update : (All Time,Yearly,Monthly,Weekly,Daily,Custom)
- Produk : CRUD Tambah Produk + Informasikan Inputannya yg relevan + Aksi
- Kategori : CRUD Tambah Kategori + Informasikan Inputannya yg relevan + Aksi
- Member / User CRUD Tambah Member + Informasikan Inputannya yg relevan + Aksi
- Terdapat fitur universal.

Transaksi
- Transaksi CRUD Status : Paid  | Proses | Done | Cancel | All | Unpaid
- Informasi Detail Transaksi Berisikan : Action | Catatan Admin (Admin bisa melakukan Input) | Kode Transaksi | Tanggal & Waktu | Nama Pesanan + Varian | Harga | Kategori | Member / Non-Member | Metode Pembayaran | Status | Bendera + IP Address 
- Data transaksi : jika di klik sesuai status = muncul status itu aja agar focus
- Terdapat fitur universal.

Admin Homepage Setting (CRUD) : Buat Setting Tampilan Homepage
- Umum (Homepage) : Title Website, Meta Deskripsi , Meta Keywords , Meta tag khusus, Footer , Sitemap.xml , Robots.txt
- Review : CRUD Tambah Review + Informasikan Inputannya yg relevan + Aksi
- FAQ : CRUD Tambah FAQ + Informasikan Inputannya yg relevan + Aksi
- Liputan Media Online : CRUD Tambah Media + Informasikan Inputannya yg relevan + Aksi
- Social Media : CRUD Tambah Sosmed + Informasikan Inputannya yg relevan + Aksi
- Informasi (Buat Halaman) : CRUD Tambah Informasi + Informasikan Inputannya yg relevan + Aksi
- Caching Otomatis : ON / OFF + Waktu

Integrasi
- Payment Gateway : iPaymu , Midtrans : Settingan On / Off + (Settingan API key + Secret Key) + Waktu Callback
- API Games : Digiflazz : Settingan On / Off + (Settingan API key + Secret Key) + Waktu Callback

Notifikasi
- Whatsapp = ON / OFF + (Settingan API key + Secret Key) + Settingan Pesannya dari setiap kondisi.
- Email = ON / OFF (CRUD) + Settingan SMTP + Settingan Pesannya dari setiap kondisi.

Proteksi
- Rate Limitting (CRUD) : ON / OFF + Limitasi
- Limitasi : Percobaan Login (1 Menit = 5x) + Register (60 Menit = 1x) + Request Lupa Password (10 Menit = 1x)
- Limitasi : Melakukan Pemesanan : Contoh 1 Menit = 2x

Log Activity (Real menggunakan API) Bukan Data Dummy
- Bendera + IP Address | Event | IP Address | Email User / Visitor | Browser | Device | Halaman | Timestamp

Auto Delete
- Auto Delete Log Activity (Support Setting Waktu : 2x24Jam) : ON / OFF
- Auto Delete Transaksi UNPAID = (Support Setting Waktu : 2x24Jam) : ON / OFF 

Tambahan
- Level Harga Membership : Visitor | Reseller | Reseller VIP | Reseller VVIP
- Auto Sync Produk + Harga dari API
- Auto Cek ID Game
- Deposit Saldo
- Support Produk Manual
- Auto Refund Saldo Jika Pembelian Gagal / Cancel

Penting (Pengiriman manual)
- Fitur kirim pesanan dari Dashboard Admin :Kirim Produk Digital Link / Akun Premium
Contoh : Link / Username / Email / Password / PIN / Catatan / Instruksi Lainnya.
Nantinya akan dikirim by email.

Sisanya : Tambahkan fitur yg selayaknya wajib ditambahkan dalam website tipe seperti ini agar sesuai standard.
